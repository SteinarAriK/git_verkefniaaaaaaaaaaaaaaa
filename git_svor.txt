#git Spurningar og svör
1. Hvað gera eftirfarandi Linux skipanir?
	cd - Skiptir um directory.
	cd ~ - Fer með þig á home directoryið.
	ls - Sjá allar skrár í directoryinu sem þú ert í.
	pwd - Prentar út directory sem þú ert í.
	mkdir - Býr til nýtt directory.

2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone - Afritar skrá yfir í möppu en geymir hana líka á upprunalegum stað.
	git log - Sýnir allt sem þú ert búin/n að uppfæra.
	git status - Synir statusinn á skránni
	git diff - Sýnir hverju hefur verið breytt í hverri uppfærslu.
	git checkout - Switch branches or restore working tree files

3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff - Sýnir hverju hefur verið breytt í hverri uppfærslu.
	b) git diff --staged - Þá ertu að skoða diffið á staged.
	c) git diff commit1 commit2 - Þetta notar maður til að pulla.

4. Hvað er útgáfustýring (e.version control)?
	Útgáfustýring heldur utan um allar breytingar svo þú getur breytt í eldri útgáfur ef þú vilt.
5. Hverjir eru helstu kostir við að nota GIT?
	Það er rosa þægilegt og auðvelt að nota.
6. Hversu oft telur þú að eigi að gera commit í verkefni, rökstuddu?
	Meira commit, meira gott :).
7. Hvað er "Working directory"?" "Staging area"?" og "Repository" í GIT?
	direktorían sem þú ert að vinna í er "working directory", "staging area" er þar sem þú geymir skránar áður en þær eru notaðar. og "repository" er data structure sem geymir öll gögn sem þú ert með.
8. Hvenær er sniðugt að nota greinar (branches)? 
	Þegar þu vilt prufa að breyta einhverju sem þú ert ekki alveg viss um.
9. Hver fann upp GIT?
	Linus Torvalds.
